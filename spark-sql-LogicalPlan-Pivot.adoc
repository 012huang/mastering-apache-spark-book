== [[Pivot]] Pivot Unary Logical Operator

`Pivot` is a link:spark-sql-LogicalPlan.adoc#UnaryNode[unary logical operator] that represents link:spark-sql-RelationalGroupedDataset.adoc#pivot[pivot] operator.

[source, scala]
----
val visits = Seq(
  (0, "Warsaw", 2015),
  (1, "Warsaw", 2016),
  (2, "Boston", 2017)
).toDF("id", "city", "year")

val q = visits
  .groupBy("city")
  .pivot("year")
  .count()
scala> q.queryExecution.logical
res0: org.apache.spark.sql.catalyst.plans.logical.LogicalPlan =
Pivot [city#8], year#9: int, [2015, 2016, 2017], [count(1) AS count#24L]
+- Project [_1#3 AS id#7, _2#4 AS city#8, _3#5 AS year#9]
   +- LocalRelation [_1#3, _2#4, _3#5]
----

`Pivot` is created when `RelationalGroupedDataset` link:spark-sql-RelationalGroupedDataset.adoc#toDF[creates a DataFrame for an aggregate operator].
